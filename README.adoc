== ActiveMQ Artemis Recipes ==

This project provides sample recipes for usage of ActiveMQ Artemis JMS broker.

=== Requirements: ===
* Apache Maven version 3
* WildFly 10
* ActiveMQ Artemis 1.0.2
* JDK 8
